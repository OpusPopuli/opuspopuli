# Commonwealth Labs — Brand & Media Kit

Complete visual identity system for Commonwealth Labs.

## Contents

### Logos (`/logos`)
All logos available in both **SVG** (vector, infinite scaling) and **PNG** (high-resolution raster).

- **Horizontal Lockup** — Use for headers, banners, wide spaces
  - `svg/cwlabs-horizontal-light.svg` + `png/cwlabs-horizontal-light.png`
  - `svg/cwlabs-horizontal-dark.svg` + `png/cwlabs-horizontal-dark.png`

- **Mark Only** — Use when space is constrained or brand is established
  - `svg/cwlabs-mark-light.svg` + `png/cwlabs-mark-light.png`
  - `svg/cwlabs-mark-dark.svg` + `png/cwlabs-mark-dark.png`

### Favicons (`/favicons`)
Available in both SVG and PNG formats:
- `favicon-16.svg` / `favicon-16.png` — 16×16 browser tab icon
- `favicon-32.svg` / `favicon-32.png` — 32×32 browser tab icon
- `favicon-512.svg` / `favicon-512.png` — 512×512 high-resolution icon

### Social Media (`/social`)
Available in both SVG and PNG formats:
- `twitter-header.svg` / `twitter-header.png` — 1500×500 Twitter/X header
- `linkedin-banner.svg` / `linkedin-banner.png` — 1584×396 LinkedIn banner
- Profile pictures: Use `favicon-512.png` for best quality

## Brand Colors

### Primary
- **Slate Dark** — `#2C3E50` (RGB: 44, 62, 80)
  - Use for: Primary text, logos, dark backgrounds

### Accent
- **Sage Green** — `#7F9C8E` (RGB: 127, 156, 142)
  - Use for: Highlights, accents, CTAs

### Backgrounds
- **Light** — `#FAFAFA` (RGB: 250, 250, 250)
- **Dark** — `#1E1E1E` (RGB: 30, 30, 30)

## Typography

### Primary: Inter
- **Display/Headings** — Bold 700
- **Subheadings** — Regular 400 (with generous letter-spacing)
- **Body Text** — Regular 400

Download: [Google Fonts](https://fonts.google.com/specimen/Inter)

## Usage Guidelines

### Do
- Use provided logo files without modification
- Maintain clear space around logo (minimum 2× mark height)
- Use slate dark (#2C3E50) or white for logo colors
- Scale logos proportionally
- Choose appropriate lockup for context

### Don't
- Alter logo colors or add gradients
- Rotate, skew, or distort the logo
- Outline the logo or add effects
- Place logo on busy backgrounds
- Recreate or redraw the logo
- Use the mark and wordmark at different scales

## Quick Start

1. **Website Header** — Use `cwlabs-horizontal-light.svg`
2. **Social Profile** — Use `favicon-512.png`
3. **Presentations** — Use `cwlabs-horizontal-light.svg` or `-dark.svg`
4. **Print** — Use SVG files for infinite scalability
