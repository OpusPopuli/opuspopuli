## Open Source License (AGPL-3.0)

The default license for QCKSTRT is the [GNU Affero General Public License v3.0](LICENSE).

Under AGPL-3.0, you can freely:
- Use the platform for any purpose
- Modify the source code
- Distribute the platform
- Run it as a network service

**Requirements:**
- Disclose source code of modifications
- License modifications under AGPL-3.0
- Provide source code to users accessing the service over a network
- Preserve copyright notices and license information

This license is ideal for:
- Non-profit organizations
- Government agencies
- Educational institutions
- Open source projects
- Organizations comfortable with source code disclosure

## Dual Licensing FAQ

### Can I use AGPL-3.0 for commercial purposes?

Yes, but you must comply with AGPL-3.0 requirements, including making your source code available to users who access your service over a network.

### When do I need a commercial license?

You need a commercial license if you:
- Cannot or don't want to disclose your source code
- Want to embed the platform in a proprietary product
- Need warranty or indemnification coverage
- Require dedicated support with SLA guarantees

### Can I switch from AGPL-3.0 to commercial later?

Yes. You can start with AGPL-3.0 and upgrade to a commercial license at any time. Contact us for migration assistance.

### What about contributions I make under AGPL-3.0?

Contributions to the upstream repository are made under AGPL-3.0 with a Contributor License Agreement (CLA). This allows Commonwealth Labs to offer commercial licensing while keeping the project open source.

### Does the commercial license apply to my custom code?

The commercial license covers the QCKSTRT platform code. Your custom code (region providers, custom features) can be licensed however you choose.

### Can I get a trial commercial license?

Yes. We offer 90-day evaluation licenses for organizations considering commercial licensing. Contact us for details.

## Contributor License Agreement

Contributors to the QCKSTRT platform sign a CLA that:
- Grants Commonwealth Labs the right to sublicense contributions
- Allows Commonwealth Labs to offer commercial licenses
- Preserves the contributor's rights to their contributions
- Ensures the project remains open source under AGPL-3.0

This dual-licensing model:
- Keeps the platform 100% open source
- Funds ongoing development and maintenance
- Provides options for commercial users
- Supports the civic technology mission
